# ConvoAI - AI Assistant Conversation Analysis

## Overview
This project analyzes AI assistant conversations and automatically generates actionable insights to improve performance.

## Key Features
- Detects user frustration patterns
- Identifies assistant response gaps
- Evaluates product recommendation effectiveness
- Highlights repeated unresolved queries
- Generates structured insights automatically

## Project Structure
- analysis/ → core logic
- data/ → input JSON files
- output/ → generated report

## How to Run
pip install -r requirements.txt  
cd analysis  
python main.py  

## Output
output/report.md
